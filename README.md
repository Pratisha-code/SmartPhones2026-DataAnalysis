# 📱 Smartphone Market Analysis Dashboard

## 📌 Project Title

Smartphone Market Analysis Using Excel, SQL, Python, and Power BI

## 📝 Brief One-Line Summary

This project analyzes smartphone market data to identify trends in pricing, battery capacity, specifications, and brand performance using data analytics tools.

## 📖 Overview

The smartphone industry is highly competitive, with numerous brands offering devices with different specifications and price ranges. This project aims to analyze smartphone data to uncover market trends, compare brand performance, and provide valuable insights through interactive visualizations.

## ❓ Problem Statement

Consumers often face difficulties in selecting smartphones due to the large number of options available in the market. Businesses and analysts need data-driven insights to understand pricing strategies, feature distributions, and market trends.

## 📊 Key Insights

- Most smartphones in the dataset support 5G connectivity.
  
- Premium smartphones generally have higher Specification Scores.
  
- Some brands offer better Value for Money (VFM) than others.
  
- Smartphones with higher RAM tend to have higher prices.
  
- Battery capacities vary significantly across different brands.
